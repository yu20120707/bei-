# 广东中考词库说明

## 广东中考课标词汇（1600+）

- 文件：`guangdong-zhongkao-1600.tsv`
- 数量：1603 个词条（保留开源词典 `zk` 标签的完整集合，故标为“1600+”而不是虚报为精确 1600）。
- 数据基础：[ECDICT](https://github.com/skywind3000/ECDICT) 的 MIT 许可词典数据；其 `tag` 字段标有 `zk`（中考）词汇。
- 中文释义优先复用本项目原有初中词库；缺少的词条使用 ECDICT 释义并删除无关领域标签。

## 广东／中山中考高频核心 800 词

- 文件：`guangdong-zhongkao-core-800.tsv`
- 数量：800 个词条。
- 其中 759 词来自上述 `zk` 范围，删除冠词、代词、基础时间词、月份、数字和典型小学生活词；按 ECDICT 的现代语料库与 BNC 词频排序。
- 另有 41 词用于补足近年广东中考阅读、完形与写作常见主题，例如 `relationship`、`responsible`、`opportunity`、`pollution`；其筛选参考了公开的[广东中考近年高频主题词汇整理](https://sjds.net/651520.html)。
- 这是项目的“高频筛选版”，不是中山市教育部门单独发布的官方 800 词表；中山英语笔试采用广东省统一命题范围。
